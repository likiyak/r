গুগল শিট কানেকশন গাইড (Setup Panel)

এই ফর্মের সকল রিয়েল এস্টেট লিড সরাসরি আপনার পার্সোনাল গুগল স্প্রেডশিটে জমা করতে নিচের পদ্ধতিটি অনুসরণ করুন।

কানেকশন করার সহজ ধাপসমূহ

ধাপ ১: একটি নতুন Google Sheet তৈরি করুন এবং প্রথম লাইনে (Row 1) নিম্নলিখিত কলামের শিরোনামগুলো হুবহু ক্রমানুসারে লিখুন:

Timestamp, Name, Phone, Email, CurrentAddress, Purpose, PropertyType, LocationAddress, BudgetPrice, SizeQty, FollowUpNotes

ধাপ ২: গুগল শিটের ওপরের মেনু থেকে Extensions > Apps Script নির্বাচন করুন।

ধাপ ৩: সেখানে থাকা পুরোনো কোড মুছে ফেলে ডান পাশের কপি করা কোডটি পেস্ট করুন এবং সেভ আইকনে ক্লিক করে Deploy > New Deployment-এ যান।

ধাপ ৪: সিলেক্ট টাইপ আইকনে ক্লিক করে Web App সিলেক্ট করুন। Execute as: Me এবং Who has access: Anyone নির্ধারণ করে Deploy চাপুন।

ধাপ ৫: গুগল থেকে অ্যাক্সেস অনুমোদন (Authorize Access) সম্পূর্ণ করে প্রাপ্ত **Web App URL**টি কপি করে নিচের বক্সে পেস্ট করে সেভ করুন।

Apps Script Code 
function doPost(e) {
  try {
    var doc = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = doc.getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    sheet.appendRow([
      new Date(),
      data.fullName || "",
      data.phoneNumber || "",
      data.email || "",
      data.currentAddress || "",
      data.purpose || "",
      data.propertyType || "",
      data.locationAddress || "",
      data.budgetPrice || "",
      data.sizeQty || "",
      data.followUpNotes || ""
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"status": "success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({"status": "error", "message": err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
