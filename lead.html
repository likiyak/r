ধাপ ১/৪: প্রাথমিক তথ্য

যোগাযোগের প্রাথমিক তথ্য

অনুগ্রহ করে আপনার সঠিক তথ্য দিন যাতে আমাদের প্রতিনিধি দ্রুত যোগাযোগ করতে পারেন।

আপনার মূল উদ্দেশ্য নির্বাচন করুন

এভারগ্রিন গ্রুপের কোন সার্ভিসের সাথে আপনি যুক্ত হতে চান?

কাঙ্ক্ষিত জমির বিবরণ (ক্রেতা)

আপনার স্বপ্নের বা প্রকল্পের জমির সঠিক ক্যাটাগরি ও এরিয়া নির্বাচন করুন।

বিক্রয়যোগ্য জমির বিবরণ (বিক্রেতা)

আপনার জমিটির দলিলাদি এবং আনুমানিক ডিমান্ডের তথ্য দিন।

খামার প্রশিক্ষণ কোর্স নির্বাচন (কৃষি উইং)

বাংলাদেশ পোল্ট্রি অ্যান্ড ফিশ ফার্মার্স অ্যাসোসিয়েশন অনুমোদিত ব্যবহারিক কোর্স।

পল্লী চিকিৎসা প্রশিক্ষণ (মেডিকেল উইং)

বাংলাদেশ পল্লী চিকিৎসা সেন্টারের অধীনে ক্যারিয়ার ও দক্ষতা উন্নয়নমূলক কোর্স।

দেশবন্ধু ফিড ডিস্ট্রিবিউটরশিপ ও অর্ডার

উচ্চ-মানের মৎস্য ও পোল্ট্রি সুষম খাদ্যের বাণিজ্যিক ডিস্ট্রিবিউশন জোন নির্বাচন।

স্মার্ট ফলো-আপ প্রশ্নাবলী (ক্রেতা)

আপনাকে সর্বোত্তম আইনানুগ পরামর্শ দিতে আমাদের এই তথ্যগুলো প্রয়োজন।

স্মার্ট ফলো-আপ প্রশ্নাবলী (বিক্রেতা)

জমির আইনি কাগজপত্র ও বিক্রয়ের ত্বরান্বিত প্রক্রিয়া নিশ্চিত করতে দিন।

ফলো-আপ ও ক্লাস শিডিউল সুবিধা

ভর্তি প্রক্রিয়া ও সেশন নির্বাচন সুগম করতে সাহায্য করুন।

ডেলিভারি ও টেকনিক্যাল সাপোর্ট

ফিড ডিস্ট্রিবিউশন প্রক্রিয়া দ্রুততর করতে বিবরণ দিন।

গুগল শিট কানেকশন গাইড (Setup Panel)

এই ফর্মের সকল লিড সরাসরি আপনার পার্সোনাল গুগল স্প্রেডশিটে জমা করতে নিচের পদ্ধতিটি অনুসরণ করুন।

কানেকশন করার সহজ ধাপসমূহ

ধাপ ১: একটি নতুন Google Sheet তৈরি করুন এবং প্রথম লাইনে (Row 1) নিম্নলিখিত কলামের শিরোনামগুলো হুবহু ক্রমানুসারে লিখুন:

Timestamp, Name, Phone, Email, CurrentAddress, Purpose, DetailCategory, SpecDetails, Financials, DynamicQty, ExtraNotes

ধাপ ২: গুগল শিটের ওপরের মেনু থেকে Extensions > Apps Script নির্বাচন করুন।

ধাপ ৩: সেখানে থাকা পুরোনো কোড মুছে ফেলে ডান পাশের কপি করা কোডটি পেস্ট করুন এবং সেভ আইকনে ক্লিক করে Deploy > New Deployment-এ যান।

ধাপ ৪: সিলেক্ট টাইপ আইকনে ক্লিক করে Web App সিলেক্ট করুন। Execute as: Me এবং Who has access: Anyone নির্ধারণ করে Deploy চাপুন।

ধাপ ৫: গুগল থেকে অ্যাক্সেস অনুমোদন (Authorize Access) সম্পূর্ণ করে প্রাপ্ত **Web App URL**টি কপি করে নিচের বক্সে পেস্ট করে সেভ করুন।

Apps Script Code 
function doPost(e) {
  try {
    var doc = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = doc.getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    
    sheet.appendRow([
      new Date(),
      data.fullName || "",
      data.phoneNumber || "",
      data.email || "",
      data.currentAddress || "",
      data.purpose || "",
      data.detailCategory || "",
      data.specDetails || "",
      data.financials || "",
      data.dynamicQty || "",
      data.extraNotes || ""
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"status": "success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({"status": "error", "message": err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
